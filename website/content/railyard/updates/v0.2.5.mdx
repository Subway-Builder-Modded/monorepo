<ChangelogSection type="features">

- Added the ability to filter compatible/incompatible/test content on **Browse** via the sidebar.

</ChangelogSection>

<ChangelogSection type="improvements">

- Added an attempt to auto-resolve the new Windows executable path to avoid wiping subscriptions
- Added support for multiple imports in library
  - Conflicting assets can be batch replaced
- Invalid imports now enumerate which files are missing instead of saying a generic "missing one or more required files" message

</ChangelogSection>

<ChangelogSection type="bugfixes">

- Fixed an issue where the `Flatpak` mount for Linux version detection wouldn't launch due to location or would become orphaned due to improper process killing
- Fixed a bug with **Enable Dev Tools** on Linux by passing the variable into flatpak-spawn

</ChangelogSection>
