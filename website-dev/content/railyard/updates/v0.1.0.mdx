Welcome to Railyard, the all-in-one manager for Subway Builder community-made content! Railyard is a free, open-source desktop app that makes it easy to discover, install, and manage community-made content for Subway Builder. No more digging through Discord threads or manually dragging files into folders - just browse, click, and play.

_Railyard is still in beta, so please help us out by reporting any problems you have on the GitHub repository!_

## For Modders & Map Creators

Want your project in Railyard? Host your files on GitHub, fill out a submission form, and a maintainer will review it.

And to everyone in the community who has been creating maps, building mods, testing early builds, and giving feedback — thank you. This project is for you! :heart:
