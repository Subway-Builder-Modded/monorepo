<ChangelogSection type="features">

- Added support for `buildings_index.bin` (post 1.3.0 map building index format)
  - Railyard now loads the `.bin` or `.json` index conditionally depending on the user's installed game version
- Added country-name search (including endonyms such as 日本, Eesti, etc.) for maps

</ChangelogSection>

<ChangelogSection type="improvements">

- Persist cache of asset versions to disk and use conditional fetches via ETags to reduce Github API request churn
- Replaced large PNG/JPG with WebP on `registry` and moved app-unused `grid/base_map` files to a separate branch to reduce refresh/clone size
  - Total clone size should now be around ~200 MB rather than ~1.4 GB

</ChangelogSection>

<ChangelogSection type="bugfixes">

- Fixed issue where newly released assets without passing integrity could be installed via update
- Fixed game version dependency parsing to honor Custom JSON version expressions over `manifest.json` when provided

</ChangelogSection>
