<ChangelogSection type="features">

- Users may now provide a list of `.osm.pbf` files that will be merged by Depot before performing extraction
- Users may now control whether military bases are colored like airports or not via a new boolean parameter: `color_military_like_aerodrome`
- Users may now set a maximum tile size in KB for buildings via a new parameter `max_building_tile_size` (to ensure that, after buildings are merged into the tiles, it does not exceed Subway Builder's 500 KB limit)

</ChangelogSection>

<ChangelogSection type="improvements">

- Added language selection for road names

</ChangelogSection>
