This guide will help you troubleshoot common issues users have with the Railyard app. If you are experiencing an issue not covered here, please reach out to us on [Discord](https://discord.gg/jrNQpbytUQ) or raise an issue on [GitHub](https://github.com/Subway-Builder-Modded/monorepo/issues).

| Issue                                                     | Solution                                                                                                                                                                                             |
| --------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Game Data Folder Not Recognized/Accepted**              | Before you can use Railyard, you must have both launched the game and launched a map in-game. This will populate the `cities/` folder within your `metro-maker4` directory, which Railyard requires. |
| **GitHub Request Error (Registry Refresh Fail)**          | See the [GitHub Token guide](/railyard/docs/v0.2/github-token).                                                                                                                                      |
| **Auto-Updater Not Detecting New Versions**               | See the [GitHub Token guide](/railyard/docs/v0.2/github-token).                                                                                                                                      |
| **Maps Fail to Fully Load In-Game**                       | You likely did not launch Subway Builder via the Railyard app. To play modded content, you must launch the game through Railyard.                                                                    |
| **Custom Content Isn't Registered In-Game**               | You likely did not enable the **Railyard Map Loader** in the in-game Mod Manager.                                                                                                                    |
| **Country Flags Are Appearing As Unicode Text (Windows)** | See the [Country Flag Emojis guide](/railyard/docs/v0.2/country-flag-emojis).                                                                                                                        |
