By default, when you submit content to the Registry, your GitHub display name and profile will be used for attribution. However, you may want to customize this information to display a different name or link.

On the [Registry's GitHub repository](https://github.com/Subway-Builder-Modded/registry), navigate to **Issues** > **New Issue** > **Update Author Profile**. Here, you can submit your author attribution update request, which will be reviewed by a Subway Builder Modded maintainer.
