This guide will walk you through using the [Markdown Playground](/registry/markdown-playground) and its various features, including how to use templates.

Markdown Playground is a tool for creating and previewing markdown content with live rendering. It supports standard markdown syntax as well as inline HTML. The playground is designed to help content creators create rich markdown content for their projects, such as map descriptions, mod documentations, etc.

## Usage {#usage}

To use the Markdown Playground, simply navigate to [the page](/registry/markdown-playground) and start typing your markdown content in the editor.

### Toolbar {#toolbar}

<p align="center">
  <Image width="1800" alt="image" src="/images/registry/docs/markdown-playground/toolbar.png" />
</p>

This toolbar provides quick access to various Markdown formatting options. You can use it to easily apply headings, bold or italic text, blockquotes, code blocks, and more without manually typing the Markdown syntax.

On the right side of the toolbar, you can switch between the text editor and the preview as well as export in either MD or HTML format.

:::tip

Your work is stored in the browser's local storage, so you can safely navigate away and return later without losing your content.

:::

## Templates {#templates}

The Markdown Playground includes a collection of templates for common content types, such as map descriptions. These templates provide a structured starting point for your content, with placeholders and formatting already set up. You can browse the available templates and select one to load it into the editor.

To get started, click on **Browse Templates** in the heading of the page. Here, you will see a popup allowing you to browse through all the available templates.

<p align="center">
  <Image
    width="900"
    alt="image"
    src="/images/registry/docs/markdown-playground/browse-templates.png"
  />
</p>

Once you select a template, you can view the description provided by the creator as well as available versions of that template. You can click **Preview Template** to view a preview of the template content before loading it into the editor.

<p align="center">
  <Image width="900" alt="image" src="/images/registry/docs/markdown-playground/preview.png" />
</p>

:::note

Clicking on a previous version of a template will load that version in the preview and allow you to use it, however it is recommended for most users to use the latest version of a template to ensure you have the most up-to-date formatting and placeholders.

:::

Once you click **Use Template**, the template content will be loaded into the editor, and you can start editing it to fit your needs.

:::danger

Using a template will overwrite any existing content in the editor, so make sure to copy and save any important content before loading a template.

:::
